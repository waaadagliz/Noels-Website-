JS Lesson 3: Data Types

JavaScript has several data types like:

  • String - text like "Hello"
  • Number - numeric values like 42
  • Boolean - true or false
  • Null - empty value
  • Undefined - a variable without a value

// Data Types Example
let username = "Noel";         // String
let age = 18;                  // Number
let isStudent = true;          // Boolean
let score = null;              // Null
let grade;                     // Undefined

let result = 
  "String: " + username + "\n" +
  "Number: " + age + "\n" +
  "Boolean: " + isStudent + "\n" +
  "Null: " + score + "\n" +
  "Undefined: " + grade;

document.getElementById("output3").textContent = result;
