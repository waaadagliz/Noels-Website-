➕ Add More Pages to Your Website

1. Create New Pages

Create files like about.html and contact.html in your project folder.

2. Add Content to Those Pages

<!-- about.html -->
<h1>About Us</h1>
<p>Welcome to Noel Codes. This is our about page.</p>

About Us

Welcome to Noel Codes. This is our about page.

3. Link Pages Using <a> Tags

<a href="about.html">Go to About Page</a>
Go to About Page

4. Organize Files (Optional)

You can create a pages folder and move your HTML files inside:

pages/
 ┣ about.html
 ┣ contact.html

Then change links like this:

<a href="pages/about.html">About</a>
