🌐 How to Create a Navigation Bar

In this tutorial, you'll learn how to build a simple navigation bar using HTML and CSS — the foundation of every website!

📄 Step 1: HTML

<nav class="navbar">
  <a href="#">Home</a>
  <a href="#">About</a>
  <a href="#">Courses</a>
  <a href="#">Contact</a>
</nav>

🎨 Step 2: CSS

.navbar {
  background-color: #333;
  padding: 10px;
  display: flex;
  gap: 20px;
}

.navbar a {
  color: white;
  text-decoration: none;
  font-weight: bold;
}
