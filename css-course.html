CSS Lesson 1: What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML elements should be displayed on screen.

3 Ways to Add CSS:

  • Inline: In the tag - <p style="color:red;">Red</p>
  • Internal: In the <style> tag in head
  • External: Best practice. Linked CSS file.
<link rel="stylesheet" href="style.css">

Example:

body {
  background-color: #f9f9f9;
  font-family: Arial, sans-serif;
}

CSS Lesson 2: Selectors & Colors

CSS selectors allow you to target HTML elements and apply styles like colors, fonts, and layouts.

Types of Selectors:

  • Element Selector: Targets all HTML elements of that type
  • Class Selector: Targets elements with a specific class (starts with .)
  • ID Selector: Targets a unique element with an ID (starts with #)

Examples:

1. Element Selector

p {
  color: blue;
}

2. Class Selector

.highlight {
  color: red;
  font-weight: bold;
}

3. ID Selector

#main-title {
  font-size: 24px;
  color: green;
}

Color Syntax:

  • color: blue; – named color
  • color: #3498db; – HEX code
  • color: rgb(255, 0, 0); – RGB

CSS Lesson 4: Borders, Margins & Padding

These three properties help you control space and appearance around and inside your HTML elements.

1. Border

2. Padding

  padding: 20px;

This adds space inside the border.

3. Margin

  margin: 30px;

This adds space outside the element (between other elements).

CSS Lesson 5: Box Model & Width/Height

Everything in CSS is a box. The box model describes how an element is structured with content, padding, borders, and margins.

CSS Box Model Structure:

  • Content: The actual text or image inside the box
  • Padding: Space around the content
  • Border: The line surrounding the padding
  • Margin: Space outside the border (between elements)

CSS Lesson 7: Display, Block, Inline & Visibility

The display property defines how elements are rendered in the document flow.

Types of Display:

  • block – Takes the full width (starts on a new line)
  • inline – Takes only as much width as needed (stays on the same line)
  • none – Hides the element completely
  • inline-block – Acts like inline but allows setting width/height

Visibility Property:

  • visible – The element is shown
  • hidden – The element is invisible but still takes up space

CSS Lesson 9: Flexbox – Making Layouts Easy

Flexbox helps align items in rows or columns and space them evenly. It’s great for building menus, galleries, or any responsive layout.

1. Basic Flexbox Container

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

2. Key Properties

  • justify-content: Aligns items horizontally
  • align-items: Aligns items vertically
  • flex-direction: Row or column layout

CSS Lesson 10: Grid – Layout Like a Pro

CSS Grid gives you full control over 2D layout – rows and columns. It’s super useful for creating structured designs.

1. Basic Grid Setup

2. Grid Items

CSS Lesson 11: Transitions & Animations

Transitions let properties change smoothly over time. Animations give full control over keyframes.

1. Transition Example

.box {
  background: #3498db;
  transition: background 0.5s ease, transform 0.3s ease;
}

.box:hover {
  background: #2ecc71;
  transform: scale(1.1);
}

2. Animation Example

@keyframes slide {
  0%   { left: 0; }
  100% { left: 100px; }
}

.animated-box {
  position: relative;
  animation: slide 2s infinite alternate;
}

CSS Lesson 12: Media Queries & Responsive Design

Media queries allow your site to adapt to different screen sizes like phones, tablets, and desktops.

1. Basic Media Query

@media (max-width: 600px) {
  .box {
    background: #e67e22;
    font-size: 14px;
  }
}

2. Responsive Example

.box {
  background: #3498db;
  color: white;
  padding: 20px;
  font-size: 18px;
}

@media (max-width: 768px) {
  .box {
    background: #9b59b6;
    font-size: 16px;
  }
}

CSS Lesson 14: Z-Index and Layering

The z-index property controls the stack order of elements — which ones appear in front or behind others.

1. Z-Index Example

.box1 {
  position: absolute;
  top: 50px;
  left: 50px;
  width: 150px;
  height: 150px;
  background: red;
  z-index: 1;
}

.box2 {
  position: absolute;
  top: 100px;
  left: 100px;
  width: 150px;
  height: 150px;
  background: blue;
  z-index: 2;
}
