Learn how to adjust your layout for phones and tablets using media queries.

1. What is a Media Query?

A media query checks the screen size and applies different styles:

@media (max-width: 600px) {
  h1 {
    font-size: 20px;
    color: blue;
  }
}

@media (max-width: 600px) {
  .box {
    background: blue;
    font-size: 14px;
  }
}

3. Why It Matters

Most users view websites on phones. Responsive design makes your site easier to read, click, and use on all devices.
