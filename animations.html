⚡ Add Animations Using CSS

Animations make your site look more dynamic. Here’s an example of a bouncing box using only CSS animations:

.box { background-color: #2c7be5; width: 100px; height: 100px; margin: 2rem auto; border-radius: 10px; animation: bounce 2s infinite; } @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } }
